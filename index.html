5. Different font Sizes

  • This is 12px - Absolute
  • This is 16px - Absolute
  • This is 1em - Relative
  • This is 1.2rem - Relative
  • This is 150% - Relative
  • This is 2vw - Relative


  • Relative: These font sizes are used to have relative font sizes. The size depends on the parental element.
  • Absolute: This font sizes never change once assigned and they are independent of size of parental element. However, they can be used to aid for using relative font sizes.
